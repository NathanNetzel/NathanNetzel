# Hi, I'm Nathan Netzel 👋

Electrical Engineering undergraduate at Londrina State University (UEL) and Technical Researcher at the NAPI SPACE program.

I work on embedded systems, aerospace instrumentation, precision positioning technologies, and real time validation environments, covering the complete development cycle from hardware design to firmware implementation and experimental validation.

## Main Areas of Interest

* Embedded Systems
* STM32 Microcontrollers
* Bare Metal Firmware
* Real Time Systems
* Precision Positioning, Navigation and Timing (PNT)
* GNSS and GNSS/INS Integration
* Hardware-in-the-Loop (HIL)
* Aerospace Instrumentation
* Data Acquisition Systems
* Kalman Filtering and Sensor Fusion
* PCB Design

## Featured Projects

### 🛰️ Modular and Scalable PNT Platform
Development of a modular Positioning, Navigation and Timing platform for high precision GNSS applications within the NAPI SPACE program.

### 🚀 Compact Onboard Computer for Sounding Rockets
Embedded avionics platform integrating MEMS sensors, Kalman filtering, flight event detection, and real time logging.

🔗 Repository:
https://github.com/NathanNetzel/Compact-Onboard-Computer-for-Sounding-Rockets

### 🔥 Low Cost DAQ System for Rocket Testing
Modular data acquisition platform for static fire testing of small rocket motors.

🔗 Repository:
https://github.com/NathanNetzel/Low-Cost-DAQ-System-for-Rocket-Testing

### 🛰️ Hardware-in-the-Loop Simulation Framework
Reusable HIL framework for validation of sounding rocket avionics.

🔗 Repository:
https://github.com/NathanNetzel/Hardware-In-the-Loop-Simulation-Framework-for-Sounding-Rocket-Onboard-Computers
