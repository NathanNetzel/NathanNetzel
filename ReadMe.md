# Hi, I'm Nathan Netzel 👋

Electrical Engineering undergraduate at Londrina State University (UEL) and Technical Researcher at the NAPI SPACE program, working on embedded systems, aerospace instrumentation, precision positioning technologies, and real time validation environments.

My work focuses on the complete development cycle of embedded electronic systems, including multilayer PCB design, bare metal firmware development, sensor integration, GNSS based navigation systems, Hardware-in-the-Loop (HIL) validation, and real time data acquisition platforms.

## Featured Projects

### 🚀 Modular and Scalable PNT Platform

Research and development of a modular and scalable Positioning, Navigation and Timing (PNT) platform within the NAPI SPACE program. The project focuses on deterministic GNSS acquisition, timing synchronization, embedded processing, Hardware-in-the-Loop validation, GNSS/INS integration, and future UAV applications.

### 🚀 Compact Onboard Computer for Sounding Rockets

Development of a compact onboard computer platform for sounding rockets, integrating MEMS sensors, embedded Kalman filtering, flight event detection, and real time data logging.

Repository:
Compact-Onboard-Computer-for-Sounding-Rockets

### 🔥 Low Cost DAQ System for Rocket Testing

Development of a modular data acquisition system for static fire testing of small rocket motors, integrating load cells, pressure sensors, signal conditioning, embedded processing, and LabVIEW monitoring tools.

Repository:
Low-Cost-DAQ-System-for-Rocket-Testing

### 🛰️ Hardware-in-the-Loop Simulation Framework

Development of a reusable HIL framework for validation of sounding rocket avionics through sensor emulation, flight simulation, telemetry, and deterministic testing environments.

Repository:
Hardware-In-the-Loop-Simulation-Framework-for-Sounding-Rocket-Onboard-Computers

## STM32 Driver Libraries

This GitHub also hosts several reusable STM32 HAL driver libraries developed during research and hardware development activities:

* ADXL357 Accelerometer
* ADXL375 Accelerometer
* ASM330 IMU
* ICM42670 IMU
* MMC5983 Magnetometer
* MS5607 Barometer
* MS5611 Barometer
* PGA308 Signal Conditioner
* MX25V8035 NOR Flash
* AT45DB641E NOR Flash

All drivers are written in C using STM32 HAL and SPI/UART interfaces, with Doxygen documentation and example implementations.
